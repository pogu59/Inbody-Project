# 🩺 InBody Simulator (인바디 시뮬레이터)

Tkinter를 활용하여 **키와 몸무게를 입력하면 BMI 계산, 체중 관리, 식단 추천까지 제공하는 GUI 프로그램**입니다.

---

## ✅ 주요 기능

- **BMI 검사**
  - 키(cm), 몸무게(kg)를 입력하여 BMI 지수 계산
  - 저체중 / 정상 / 과체중 / 비만 판정 결과 출력

- **체중 관리**
  - 표준 체중 계산
  - 목표 체중까지 예상 기간(개월)
  - 하루 권장 섭취 칼로리(kcal) 제공

- **식단 추천**
  - BMI 상태에 따라 1900 / 2100 / 2300 kcal 식단 추천
  - 체중 증가, 유지, 감량에 맞춘 식단 제공
  - 운동 추천 메시지 출력 (웨이트 / 유산소)

- **기능 버튼**
  - 다시 입력 (초기화)
  - 종료 버튼 (종료 확인 알림)

---

## 🖥️ 실행 화면 구성

- 상단: 프로그램 제목
- 좌측: 옵션 선택 (BMI, 체중 관리, 식단)
- 우측: 결과 출력 영역
- 하단: 다시 입력 / 종료 버튼

---
